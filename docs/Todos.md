# how todos will work in my project manager

## file layout

[todo group]
title,description,due date, done

json data:
key: 
{ todo group: [{title,description,due date, done}] }

## capabilities
- add todo
- remove todo
- change todo (title,description,due date)
- add group
- change done
- save to txt file
- save to binary file (.todo)
